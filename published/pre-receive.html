A Git Pre-Receive Hook

This will prevent pushes from a cloned repository. Still, it might be better to allow pushes. That way a feature branch that only contains work relevant to this project could be pushed and merged as normal. 

#!/usr/bin/env bash

echo ""
echo "=================================================="
echo "This repository is no longer available for pushes."
echo "=================================================="
echo ""
exit 1

rm ../.git/hooks/pre-receive
